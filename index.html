OpenTether

Bridges Android devices running a SOCKS5 proxy to your OpenWrt router via ADB port forwarding and hev-socks5-tunnel. Multi-device, hotplug-automatic, LuCI-managed.

v1.3.0-r1 OpenWrt 24+ PKGARCH all

Install from this repository

1
Add the signing key
router shell 
curl -o /etc/apk/keys/opentether.pub \
  https://helafaye.github.io/opentether/key-build.pub
Only needed once. Persists across reboots.
2
Add the repository
router shell 
echo https://helafaye.github.io/opentether/packages/all \
  >> /etc/apk/repositories
apk update
Only needed once.
3
Install
router shell 
apk add opentether luci-app-opentether
Install opentether alone if you don't use LuCI.
For OpenWrt 23.05 and earlier, use the .ipk files from the GitHub release with opkg install.
